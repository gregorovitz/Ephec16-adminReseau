Service VoIP

Docker
Les fichiers nécessaires à l'installation du service VoIP se trouve sur le Docker : mrspeculoos/wtasterisk

Configuration serveur
Les principaux fichiers de configurations d'Asterisk se trouvent dans ces trois fichiers ci-dessous :
 - /etc/asterisk/sip.conf : Configuration globale d'Asterisk
 - /etc/asterisk/users.conf : Configuration des utilisateurs
 - /etc/asterisk/extensions.conf : Configuration du Dialplan
Afin de conserver une sauvegarde nous allons créer des copies de ces fichiers en cas de soucis :
$ sudo mv /etc/asterisk/sip.conf /etc/asterisk/sip-save.conf
$ sudo mv /etc/asterisk/users.conf /etc/asterisk/users-save.conf
$ sudo mv /etc/asterisk/extensions.conf /etc/asterisk/extensions-save.conf

Information sur sip.conf
Le [general] va prendre en compte la configuration générale des SIP, tout le monde la possèdera.
Les parties [*****](!) sont des groupes. Ils vont spéciéfier la configuration d’un groupe de SIP comme par exemple les marketing. La configuration d’un groupe ne va affecter que les SIP compris dans ce groupe.
Les parties [xyz](*****) sont les SIP. “xyz” est le nom et le (*****) est le groupe.
-transport=udp : Indique quel type de transport par défaut utilisé
-language=fr : Change la langue en français
-type=friend : Le type d'utilisateur. Afin de garder le côté pratique, nous le laisserons en friend
-host=dynamic : Y mettre l'adresse IP de l'utilisateur. Si dynamic est mis, Asterix configurera automatiquement
-context=from-internal : Concerne le moment où l'appel est reçu (voir extension.conf)
-disallow=all : Refuse tous les codec exceptés ce qui est mis dans allow
-allow=ulaw : N'autorise que les codec ulaw
-secret : Est le mot de passe
-callgroup=x : Indique dans quel groupe x le SIP appartiendra
-pickupgroup=x : Indique quel groupe le SIP poura décrocher l’appel
-namedcallgroup=work : Est le nom générique du groupe

exten => 101,1,Dial(SIP/Ouvrier_1,20)

exten => 102,1,Dial(SIP/Ouvrier_2,20)
exten => 103,1,Dial(SIP/Ouvrier_3,20)

exten => 200,1,Dial(acco,20)

exten => 202,1,Dial(SIP/Compta_2,20)

Informations sur extensions.conf
[xyz] : Sont les contextes indiqués dans les groupes de sip.conf
exten => : sont les actions
xxx,n,Dial(SIP/nom,nn) : "xxx" est le numéro qu'Asterisk reçoit en appel, "n" est la priorité qu'Asterisk donne à cette action, "Dial" est la commande, "SIP/nom" est le nom d'utilisateur qu'il faut appeler dans ce cas ci, "nn" est le temps avant qu'Asterisk ne coup la tentative d'appel si pas de réponse.
include => : Indique qu'il faut "copier" les commandes du contexte indiqué.
